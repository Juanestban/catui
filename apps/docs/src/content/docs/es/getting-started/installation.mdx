## Inicio Rapido

Instalar el paquete con **npm**, **yarn** y **pnpm**

<Tabs>
<TabItem label="npx">

```sh
npx install catui-design
```

</TabItem>
<TabItem label="yarn">

```sh
yarn add catui-design
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add catui-design
```

## Uso del tema como envoltorio de CatUI

Para usarlo correctamente con los tokens que se estan usando en esta libreria, necesitaras usar `CatuiTheme`

<Tabs>
<TabItem label="typescript">
<img src="https://skillicons.dev/icons?i=ts" alt="typescript" />

`src/root.tsx`

```tsx
import { StrictMode } from 'react';
import ReactDOM from 'react-dom/client';
import { CatuiTheme } from 'catui-design';

import App from './App';

import './index.css';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <CatuiTheme>
      <App />
    </CatuiTheme>
  </StrictMode>,
);
```

</TabItem>
